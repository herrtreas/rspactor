Welcome to RSpactor. You may want to have a look at these first steps if you are new to RSpactor:

  1. Create a symlink to get the nifty command-line tool (see INSTALL included with the Disk Image)
  2. Check your Preferences (⌘,): Keep up-to-date with the Auto-Update feature
  3. Kick off: Enter the root path to your ruby project into the box above (⌘+l) and hit 'enter'
